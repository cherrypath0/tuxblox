Installation Guide

Step-by-step instructions set up TuxBlox

Important Compatibility Notice

TuxBlox is specifically built for Linux distributions. TuxBlox cannot be installed on Windows, macOS or any other environments.

System Requirements

Hardware Requirements

  • Processor: Intel or AMD, 64-bit x86 (x86_64) Architecture with SSE3 Support
  • Memory: 4 GB
  • Storage: 1 GB

Software Requirements

  • Operating System: Any major 64-bit Linux distribution
  • Kernel Version: 5.0 or higher
  • Filesystem: Unencrypted (because the compatibility layer relies on methods that require an unencrypted filesystem)
NOTICE: Still In Development

TuxBlox is still VERY early in development, so the installation is not possible yet.
